# PDFNeo - Professional PDF Editor

PDFNeo is a modern, client-side PDF editing application designed for speed, security, and ease of use. All processing happens directly in your browser, ensuring your sensitive documents never leave your device.

## 🚀 Features

- **Merge PDF**: Combine multiple PDF files into a single document.
- **Privacy First**: Local processing means no file uploads to servers.
- **Modern UI**: Smooth animations and a refined design system.

## 🛠️ Built With

- **Vite** & **React**
- **TypeScript**
- **Tailwind CSS** & **shadcn/ui**
- **pdf-lib** for client-side PDF manipulation
- **Framer Motion** for animations

## 💻 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/kushwi15/PDFNeo.git
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```
